# Mesto — Локальный социальный профиль

**Mesto** — это одностраничное веб-приложение, реализующее функциональность профиля пользователя с возможностью редактирования, добавления и удаления карточек с фотографиями.

## Функциональность

- Редактирование профиля и аватара
- Добавление и удаление карточек
- Открытие изображений в полноэкранном режиме
- Лайки на карточках
- Валидация форм
- Анимация попапов

## Технологии

- HTML5, CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- ООП-подход
- Модули JS
- Работа с DOM
